# yt-dlp Download Tool

`yt-dlp` を利用して、動画のダウンロードやMP3形式への音声抽出を簡単に行うためのPythonスクリプトです。
実行時に `yt-dlp` の自動アップデートを行うため、常に最新の状態でダウンロードを実行できます。また、MP3抽出時には曲名やアーティスト名などのメタデータ（ID3タグ）の付与が可能です。

## 前提条件 (Prerequisites)

このスクリプトを実行するには、以下の環境が必要です。

* **Python 3.x**
* **ffmpeg**: MP3への変換およびメタデータの書き込みに必須です。OSの環境変数（PATH）が通っていることを確認してください。

※ `yt-dlp` パッケージ自体は、スクリプト実行時に `pip` 経由で自動的にインストール・最新版への更新が行われます。

## 使い方 (Usage)

コマンドラインから引数を指定してスクリプトを実行します。

### 1. 基本的な動画のダウンロード

URLのみを指定した場合、デフォルトの最高画質・高音質で動画としてダウンロードします。

```bash
python yt-dlp_download.py "<動画のURL>"

```

### 2. MP3形式でのダウンロード（音声抽出）

`--mp3` オプションを指定すると、動画から音声を抽出し、MP3形式で保存します。

```bash
python yt-dlp_download.py "<動画のURL>" --mp3

```

### 3. メタデータ（タグ情報）を指定してのMP3ダウンロード

`-t` (`--title`) および `-a` (`--artist`) オプションを使用して、ファイル名やメタデータを指定できます。スペースを含む文字列（英語の曲名など）を指定する場合は、必ずダブルクォーテーション `""` で囲んでください。

```bash
python yt-dlp_download.py "<動画のURL>" --mp3 -t "曲名" -a "アーティスト名"

```

* `-t "曲名"`: 出力されるファイル名、プロパティの「タイトル」「アルバム名」に適用されます。
* `-a "アーティスト名"`: プロパティの「参加アーティスト」「アルバムアーティスト」に適用されます。

## オプション引数一覧

| 引数 / オプション | 省略形 | 説明 |
| --- | --- | --- |
| `url` | なし | ダウンロード対象のURL（必須） |
| `--mp3` | なし | 音声を抽出し、MP3形式で保存フラグ |
| `--title` | `-t` | 曲名の指定（MP3形式でのダウンロード時のみ有効） |
| `--artist` | `-a` | アーティスト名の指定（MP3形式でのダウンロード時のみ有効） |
| `--help` | `-h` | スクリプトのヘルプメッセージを表示 |

## 処理フロー

1. `pip install -U yt-dlp` を実行し、ライブラリを最新化。
2. 指定された引数に基づき、`yt-dlp` のコマンドリストを構築。
3. メタデータ指定がある場合は、`--postprocessor-args` を用いて内部的に `ffmpeg` へタグ情報を引き渡し。
4. `subprocess` モジュールを通じて安全にコマンドを実行。
